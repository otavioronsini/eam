## Metodologia

O desenvolvimento do projeto seguiu uma abordagem orgânica e sob demanda. As tarefas foram executadas de forma colaborativa por uma equipe de três desenvolvedores, atuando conforme as necessidades do aplicativo surgiam, sem a adoção de metodologias de gestão rígidas. 

O trabalho foi focado em entregas práticas, dividindo os esforços nas seguintes frentes:
*   Configuração e integração do backend utilizando Firebase.
*   Estruturação da interface de usuário (UI) e fluxos iniciais.
*   Implementação da lógica de negócios e desenvolvimento das demais funcionalidades do aplicativo.

### Ferramentas e Tecnologias Utilizadas
*   **Framework:** Flutter (Dart).
*   **Ambientes de Desenvolvimento (IDEs):** VS Code e Android Studio, englobando testes e compilação em ambientes Windows e macOS.
*   **Controle de Versão:** Git e GitHub, utilizados como repositório central para unificar o código desenvolvido por toda a equipe.
